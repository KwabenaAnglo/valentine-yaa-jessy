A special Valentine's Day message for my sister built with pure HTML, CSS, and JavaScript.

## Features

- **Animated Hero**: Elegant gold script typography with shimmer effects
- **Floating Hearts**: Continuous particle animation system
- **Love Letter**: Typewriter-style reveal animation
- **Interactive Cards**: "Reasons I Love You" with hover effects
- **CTA Animation**: Heartbeat animation with surprise interaction
- **Responsive Design**: Optimized for all devices
- **Professional Performance**: Optimized animations and smooth scrolling

## Technical Stack

- **HTML5**: Semantic markup with accessibility in mind
- **CSS3**: Modern animations, gradients, and responsive design
- **Vanilla JavaScript**: No dependencies, pure performance
- **Google Fonts**: Playfair Display & Dancing Script typography

## Getting Started

1. Clone or download the project
2. Ensure `Val.jpg` is in the root directory
3. Open `index.html` in your browser
4. Or run a local server:
   ```bash
   python -m http.server 8000
   ```

## Browser Support

- Chrome 60+
- Firefox 55+

- Edge 79+

## Performance Features

- CSS animations using GPU acceleration
- Intersection Observer for scroll-triggered animations
- Optimized font loading with preconnect
- Responsive images and lazy loading ready
- Smooth scrolling with hardware acceleration

## Customization

- Update colors in CSS variables (`:root` section)
- Modify content in `index.html`
- Adjust animation speeds in CSS keyframes
- Add new floating heart emojis in `script.js`
